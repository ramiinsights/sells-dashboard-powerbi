# 📊 Sales Dashboard Project

## 📌 Overview

This project is a **Sales Dashboard** that visualizes profit and selling data in an interactive way.
It helps to analyze performance by **Name, Product, and State**.

---

## 🎯 Objectives

* Analyze profit distribution across different people
* Identify top-performing products
* Compare profit across different states
* Track selling price trends

---

## 📊 Dashboard Features

* ✅ Name-wise Profit Analysis
* ✅ Product-wise Profit Visualization
* ✅ State-wise Profit Comparison
* ✅ Product Selling Price Trend
* ✅ Interactive Day Filter (Slider)
